# Image Assets

This directory contains static images for the frontend.

## Required Images

1. `hero-bg.jpg` - Hero section background image (1920x1080 recommended)
   - Dark, abstract or tech-themed background
   - Used in dark mode only

2. `dashboard-preview.jpg` - Dashboard preview screenshot (1920x1080 recommended)
   - Screenshot of the Paper dashboard
   - Shows document list and QA interface

## Adding Images

Place optimized images in this directory. Recommended:
- Use WebP or optimized JPEG format
- Compress images before adding (use tools like squoosh.app)
- Keep file sizes under 200KB for fast loading
